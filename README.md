# 1.2.3-Let-s-Play-Game-website
**1,2,3 Lets Play** is a responsive online games website featuring both premium and free games. Users can instantly play games, switch to dark mode, and explore interactive game cards with images, titles, and links. Built with HTML, CSS, and JavaScript for a clean, modern, and mobile-friendly gaming hub.

## Features
- Premium and Free games sections
- Interactive game cards with images, titles & play buttons
- Dark mode toggle for eye-friendly browsing
- Fully responsive and mobile-friendly layout
- Smooth hover effects and modern UI design

## Technologies
- HTML5
- CSS3
- JavaScript
